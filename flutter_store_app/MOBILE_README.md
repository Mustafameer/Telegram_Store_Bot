# تطبيق الموبايل - متجر تليجرام

تم إنشاء تطبيق موبايل محسّن للهواتف الذكية (Android و iOS) لمشروع متجر تليجرام.

## المميزات

✅ **واجهة موبايل محسّنة**
- Bottom Navigation Bar للتنقل السريع
- تصميم متجاوب يعمل على جميع أحجام الشاشات
- دعم كامل للغة العربية مع خط Cairo

✅ **الميزات الكاملة**
- عرض المتاجر والمنتجات
- إدارة الطلبات والسلة
- نظام الرسائل بين البائع والعميل
- إدارة الزبائن بالدين
- مزامنة تلقائية مع السحابة

✅ **دعم متعدد المنصات**
- Android
- iOS
- Windows/Linux/MacOS (Desktop)

## متطلبات التشغيل

- Flutter SDK 3.10.3 أو أحدث
- Android Studio (للتطوير على Android)
- Xcode (للتطوير على iOS - macOS فقط)

## تعليمات التشغيل

### 1. تثبيت المكتبات
```bash
cd flutter_store_app
flutter pub get
```

### 2. تشغيل على Android
```bash
flutter run -d android
```

### 3. تشغيل على iOS (macOS فقط)
```bash
flutter run -d ios
```

### 4. بناء APK للـ Android
```bash
flutter build apk --release
```

### 5. بناء IPA للـ iOS (macOS فقط)
```bash
flutter build ios --release
```

## البنية

التطبيق يستخدم نفس قاعدة البيانات `store_local_new.db` الموجودة في المجلد الرئيسي للبوت، مما يسمح بمزامنة البيانات بين التطبيق والبوت.

## التصميم

- **الألوان**: تصميم حديث بألوان عصرية (Teal/Premium Green)
- **الخطوط**: خط Cairo لدعم اللغة العربية بشكل مثالي
- **التصميم**: Material Design 3 مع دعم الوضع الداكن

## ملاحظات

- التطبيق يعمل بشكل مستقل عن كود البوت (Python) ولكنه يقرأ ويكتب على نفس قاعدة البيانات
- المزامنة التلقائية تعمل كل 15 دقيقة
- يمكن المزامنة اليدوية من خلال زر المزامنة في شريط التطبيق
