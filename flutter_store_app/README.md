
# تطبيق إدارة المتجر (Desktop)

تم إنشاء مشروع Flutter لمشروع المتجر المكتبي.

## متطلبات التشغيل
- Flutter SDK مثبت على الجهاز.
- نظام التشغيل Windows.

## تعليمات التشغيل
1. افتح التيرمنال وتوجه للمجلد:
   `cd flutter_store_app`

2. قم بتثبيت المكتبات:
   `flutter pub get`

3. لتشغيل التطبيق:
   `flutter run -d windows`

## ملاحظات
- التطبيق يستخدم نفس قاعدة البيانات `store.db` الموجودة في المجلد الرئيسي للبوت.
- تم استخدام تصميم Premium مع دعم الخط العربي (Cairo) وألوان عصرية.
- التطبيق منفصل تماماً عن كود البوت (Python) ولكنه يقرأ ويكتب على نفس قاعدة البيانات.

## المميزات الحالية
- عرض قائمة المتاجر (Sellers).
- الدخول لكل متجر وعرض المنتجات والطلبات والفئات.
- واجهة مستخدم حديثة.
